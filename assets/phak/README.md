# FAA PHAK source images

These 17 optimized page crops were rendered from the U.S. Federal Aviation
Administration's _Pilot's Handbook of Aeronautical Knowledge_,
FAA-H-8083-25C (2023):

https://www.faa.gov/regulations_policies/handbooks_manuals/aviation/faa-h-8083-25c.pdf

Each image retains its printed handbook page number and figure caption. The
exact source page and accessible description are also stored on the matching
`image` question in `src/content/phak.json`. Run
`scripts/content/extract_phak.py` to reproduce the crops from the pinned source.
